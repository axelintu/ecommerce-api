# ecommerce-api
Back end for ecommerce-app front end.

## Installation

Clone this repository and install dependencies
```bash  
git clone git@github.com:axelintu/ecommerce-api.git
cd ecommerce-api
npm install 
```

Copy and rename .env.example to .env and populate the variables
```bash
cp .env.example .env
```

### Start the project

For development use
```bash
npm run dev
```
